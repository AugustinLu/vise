Tutorial – band figure and density of states¶

Calculation of the band structure and the density of states¶

We can create the input files for the calculations of the band structure and the density of states by typing .. code-block:

vise vs -t band

and .. code-block:

vise vs -t dos

in each directory, we run the vasp calculations as mentioned using custodian wrapper.

Here, the band path is determined based upon the [seekpath code](https://www.materialscloud.org/work/tools/seekpath), so if one uses the plot for publication or presentation, please cite the following paper.

  • Y. Hinuma, G. Pizzi, Y. Kumagai, F. Oba, I. Tanaka, Band structure diagram paths based on crystallography, Comp. Mat. Sci. 128, 140 (2017). DOI: 10.1016/j.commatsci.2016.10.015 (arXiv version: arXiv:1602.06402).

vise also provides the plotters of BS and DOS based on with plot_band (=:code:pb) and plot_dos (=:code:pd) sub-commands. Type the following commands in band/ and dos/, respectively. .. code-block:

vise pb -f band.pdf

and .. code-block:

vise pd -f dos.pdf

The band gap is also evaluated using bg`(=:code:`band_gap) sub command. .. code-block:

vise bg -v vasprun.xml.finish -o OUTCAR.finish
